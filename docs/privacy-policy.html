2. Che dati raccogliamo

DatoScopoBase giuridicaConservazione
ID sessione anonima Contare utenti attivi (statistiche aggregate) Legittimo interesse (art. 6.1.f GDPR) 30 giorni di inattività
Email + username Account community (opzionale) Consenso esplicito (art. 6.1.a GDPR) Fino a cancellazione account
Recensioni e libri letti Funzionalità principale dell'app Esecuzione contratto (art. 6.1.b) Localmente sul dispositivo; su Supabase se sincronizzate
Stato online / piattaforma Indicatore community "utenti online" Legittimo interesse Aggiornato ogni sessione, cancellato dopo 30 giorni
Query di ricerca libri Ricerca tramite Google Books e Open Library Esecuzione contratto Non memorizzate da noi; soggette alle policy di Google e Open Library

Non raccogliamo: dati di geolocalizzazione, contatti, foto, dati biometrici, dati finanziari, dati di navigazione web esterni all'app.

3. Terze parti

  • Google Books API — ricerca libri. Le query vengono trasmesse a Google. Privacy Google
  • Open Library (Internet Archive) — ricerca libri alternativa. Termini Open Library
  • Supabase — backend community (autenticazione, dati cloud). Server in UE (Frankfurt). Privacy Supabase
  • GitHub — aggiornamenti automatici app (download APK/EXE). Privacy GitHub

4. Dati dei minori

L'app non è destinata a bambini under 13. Non raccogliamo consapevolmente dati di minori. Se sei un genitore e ritieni che tuo figlio abbia fornito dati, contattaci tramite GitHub Issues per la cancellazione immediata.

5. I tuoi diritti (GDPR)

Hai diritto a:

  • Accesso — sapere quali dati abbiamo su di te
  • Rettifica — correggere dati inesatti
  • Cancellazione — eliminare il tuo account e tutti i dati associati
  • Portabilità — esportare i tuoi dati in formato leggibile
  • Opposizione — opporti al trattamento basato su legittimo interesse
  • Revoca consenso — in qualsiasi momento, tramite Impostazioni → Privacy

Per esercitare questi diritti: apri un'issue su GitHub oppure usa la funzione "Elimina account" nelle impostazioni dell'app.

6. Sicurezza

I dati in transito sono protetti da HTTPS/TLS. I dati su Supabase sono protetti da Row Level Security (RLS): ogni utente può accedere solo ai propri dati. Le password non vengono mai memorizzate in chiaro.

7. Modifiche a questa policy

In caso di modifiche sostanziali, verrà mostrata una notifica all'interno dell'app. La data di "ultimo aggiornamento" in cima a questa pagina verrà aggiornata.
